# 📋 TaskFlow

Uma aplicação web simples e intuitiva para gerenciamento de tarefas, desenvolvida com HTML, CSS e JavaScript.

## 🚀 Acesse o Projeto

🔗 Projeto online: https://mararubiaa.github.io/taskflow/

## ✨ Funcionalidades

* Adicionar tarefas
* Visualizar tarefas em lista
* Interface simples e responsiva
* Execução diretamente no navegador
* Hospedagem em nuvem com GitHub Pages

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## 📂 Estrutura do Projeto

```

taskflow
│
├── index.html
└── README.md

```

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de participação do Vibe Build

## 🌐 Hospedagem

O projeto está hospedado utilizando GitHub Pages, permitindo acesso público por meio de um navegador web sem necessidade de instalação.
